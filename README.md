# Energy-Monitoring-System-using-IOT
Energy Monitoring System using IOT and integration with ThingSpeak

Materials Used:

1)Node MCU 1.0( 12E module)

2)Current Sensor

3)Voltage Sensor

4)Jumper Wires

5)Buzzer

6)9 Volt HW Battery
